## 1. 系统架构

### 1.1 整体流程

```
输入数据（价格 + 成本）
        ↓
  利润求解器（ProfitSolver）
        ↓
    ┌─────────┬─────────┬─────────┬─────────┐
    │ 当前利润 │ 目标利润 │ 目标利润率│ 目标ROI │
    └─────────┴─────────┴─────────┴─────────┘
        ↓
  输出结果（净利润 + 建议售价）
        ↓
    折扣模拟
        ↓
  最终报告
```

### 1.2 核心组件

| 组件 | 文件 | 功能 |
|------|------|------|
| **ProfitSolver** | `profit_solver.py` | 核心算法引擎 |

### 2.2 核心公式

#### 公式1：计算当前利润

**算法名称**：`solve_current`

**输入**：
- `sale_price`: 售价
- `variable_rate_total`: 变量费率（0-1）
- `fixed_cost_total`: 固定成本

**计算步骤**：

```python
# 步骤1：计算净利润
net_profit = sale_price * (1 - variable_rate_total) - fixed_cost_total

# 步骤2：计算净利率
net_margin = net_profit / sale_price  # 如果 sale_price = 0，则 net_margin = 0

# 步骤3：计算保本价
if variable_rate_total < 1:
    break_even_price = fixed_cost_total / (1 - variable_rate_total)
else:
    break_even_price = ∞  # 无法达到保本

# 步骤4：计算保本折扣率
if list_price > 0:
    break_even_discount_ratio = break_even_price / list_price
else:
    break_even_discount_ratio = ∞

# 步骤5：判断是否亏损
is_loss = net_profit < 0
```

**数学表达式**：

```
净利润 = 售价 × (1 - 变量费率) - 固定成本

净利率 = 净利润 / 售价

保本价 = 固定成本 / (1 - 变量费率)

保本折扣率 = 保本价 / 原价
```

**示例计算**：

```
输入：
  sale_price = 100元
  variable_rate_total = 0.35（35%）
  fixed_cost_total = 50元

计算：
  净利润 = 100 × (1 - 0.35) - 50 = 65 - 50 = 15元
  净利率 = 15 / 100 = 15%
  保本价 = 50 / (1 - 0.35) = 76.92元
  保本折扣率 = 76.92 / 120 = 64.1%

输出：
  net_profit = 15元
  net_margin = 0.15（15%）
  is_loss = false
  break_even_price = 76.92元
  break_even_discount_ratio = 0.641
```

#### 公式2：目标利润定价

**算法名称**：`target_profit_price`

**目标**：给定目标净利润，求所需售价

**输入**：
- `target_profit`: 目标净利润
- `variable_rate_total`: 变量费率
- `fixed_cost_total`: 固定成本

**公式推导**：

```
已知：
  净利润 = 售价 × (1 - 变量费率) - 固定成本

求：
  售价 = ?

解：
  目标利润 = 售价 × (1 - 变量费率) - 固定成本
  目标利润 + 固定成本 = 售价 × (1 - 变量费率)
  售价 = (目标利润 + 固定成本) / (1 - 变量费率)
```

**算法实现**：

```python
def target_profit_price(target_profit, variable_rate_total, fixed_cost_total):
    return (target_profit + fixed_cost_total) / (1 - variable_rate_total)
```

**示例计算**：

```
输入：
  target_profit = 20元
  variable_rate_total = 0.35
  fixed_cost_total = 50元

计算：
  所需售价 = (20 + 50) / (1 - 0.35)
           = 70 / 0.65
           = 107.69元

输出：
  suggested_price = 107.69元
```

#### 公式3：目标利润率定价

**算法名称**：`target_margin_price`

**目标**：给定目标净利率，求所需售价

**输入**：
- `target_margin`: 目标净利率（0-1）
- `variable_rate_total`: 变量费率
- `fixed_cost_total`: 固定成本

**公式推导**：

```
已知：
  净利率 = 净利润 / 售价
  净利润 = 售价 × (1 - 变量费率) - 固定成本

求：
  售价 = ?

解：
  目标净利率 = (售价 × (1 - 变量费率) - 固定成本) / 售价
  目标净利率 × 售价 = 售价 × (1 - 变量费率) - 固定成本
  目标净利率 × 售价 = 售价 - 变量费率 × 售价 - 固定成本
  目标净利率 × 售价 + 变量费率 × 售价 = 售价 - 固定成本
  售价 × (目标净利率 + 变量费率) = 售价 - 固定成本
  售价 - 售价 × (目标净利率 + 变量费率) = 固定成本
  售价 × (1 - 目标净利率 - 变量费率) = 固定成本
  售价 = 固定成本 / (1 - 目标净利率 - 变量费率)
```

**算法实现**：

```python
def target_margin_price(target_margin, variable_rate_total, fixed_cost_total):
    # 边界检查
    if target_margin >= 1 - variable_rate_total:
        raise ValueError("目标利润率不可达")
    
    return fixed_cost_total / (1 - variable_rate_total - target_margin)
```

**示例计算**：

```
输入：
  target_margin = 0.20（20%）
  variable_rate_total = 0.35
  fixed_cost_total = 50元

计算：
  所需售价 = 50 / (1 - 0.35 - 0.20)
           = 50 / 0.45
           = 111.11元

输出：
  suggested_price = 111.11元
```

#### 公式4：目标ROI定价

**算法名称**：`target_roi_price`

**目标**：给定目标投资回报率（ROI），求所需售价

**输入**：
- `target_roi`: 目标ROI（0-N）
- `variable_rate_total`: 变量费率
- `fixed_cost_total`: 固定成本（即投资成本）

**定义**：
```
ROI = 净利润 / 固定成本
```

**公式推导**：

```
已知：
  ROI = 净利润 / 固定成本
  净利润 = 售价 × (1 - 变量费率) - 固定成本

求：
  售价 = ?

解：
  目标ROI = (售价 × (1 - 变量费率) - 固定成本) / 固定成本
  目标ROI × 固定成本 = 售价 × (1 - 变量费率) - 固定成本
  目标ROI × 固定成本 + 固定成本 = 售价 × (1 - 变量费率)
  固定成本 × (目标ROI + 1) = 售价 × (1 - 变量费率)
  售价 = 固定成本 × (目标ROI + 1) / (1 - 变量费率)
```

**算法实现**：

```python
def target_roi_price(target_roi, variable_rate_total, fixed_cost_total):
    return fixed_cost_total * (1 + target_roi) / (1 - variable_rate_total)
```

**示例计算**：

```
输入：
  target_roi = 0.30（30%）
  variable_rate_total = 0.35
  fixed_cost_total = 50元

计算：
  所需售价 = 50 × (1 + 0.30) / (1 - 0.35)
           = 50 × 1.30 / 0.65
           = 100元

输出：
  suggested_price = 100元
```

#### 公式5：折扣模拟

**算法名称**：`simulate_discounts`

**目标**：模拟不同折扣下的利润情况

**输入**：
- `payload`: ProfitInput（原价、变量费率、固定成本）
- `ratios`: 折扣率列表（如 [0.95, 0.9, 0.85]）

**计算步骤**：

```python
def simulate_discounts(payload, ratios):
    results = []
    
    for ratio in ratios:
        # 1. 计算折扣价
        deal_price = payload.list_price * ratio
        
        # 2. 计算净利润
        net_profit = deal_price * (1 - payload.variable_rate_total) - payload.fixed_cost_total
        
        # 3. 计算净利率
        net_margin = net_profit / deal_price if deal_price > 0 else 0
        
        # 4. 判断是否亏损
        is_loss = net_profit < 0
        
        results.append({
            'discount_ratio': ratio,
            'deal_price': deal_price,
            'net_profit': net_profit,
            'net_margin': net_margin,
            'is_loss': is_loss
        })
    
    return results
```

**示例计算**：

```
输入：
  list_price = 120元
  variable_rate_total = 0.35
  fixed_cost_total = 50元
  ratios = [0.95, 0.9, 0.85]

计算：
  95折：
    deal_price = 120 × 0.95 = 114元
    net_profit = 114 × (1 - 0.35) - 50 = 24.1元
    net_margin = 24.1 / 114 = 21.1%
    is_loss = false

  90折：
    deal_price = 120 × 0.9 = 108元
    net_profit = 108 × 0.65 - 50 = 20.2元
    net_margin = 20.2 / 108 = 18.7%
    is_loss = false

  85折：
    deal_price = 120 × 0.85 = 102元
    net_profit = 102 × 0.65 - 50 = 16.3元
    net_margin = 16.3 / 102 = 16.0%
    is_loss = false

输出：
  [
    {discount_ratio: 0.95, deal_price: 114, net_profit: 24.1, net_margin: 0.211, is_loss: false},
    {discount_ratio: 0.90, deal_price: 108, net_profit: 20.2, net_margin: 0.187, is_loss: false},
    {discount_ratio: 0.85, deal_price: 102, net_profit: 16.3, net_margin: 0.160, is_loss: false}
  ]
```

## 5. 完整实现示例

### 5.1 Python实现

```python
from dataclasses import dataclass
from typing import List

@dataclass
class ProfitInput:
    sale_price: float
    list_price: float
    variable_rate_total: float
    fixed_cost_total: float

@dataclass
class ProfitResult:
    net_profit: float
    net_margin: float
    is_loss: bool
    break_even_price: float
    break_even_discount_ratio: float

@dataclass
class DiscountSimulation:
    discount_ratio: float
    deal_price: float
    net_profit: float
    net_margin: float
    is_loss: bool

class ProfitSolver:
    """V5.1 利润求解器 - 完整实现"""
    
    @staticmethod
    def solve_current(payload: ProfitInput) -> ProfitResult:
        """计算当前利润"""
        # 1. 计算净利润
        net_profit = payload.sale_price * (1 - payload.variable_rate_total) - payload.fixed_cost_total
        
        # 2. 计算净利率
        net_margin = (net_profit / payload.sale_price) if payload.sale_price > 0 else 0.0
        
        # 3. 计算保本价
        if payload.variable_rate_total < 1:
            break_even_price = payload.fixed_cost_total / (1 - payload.variable_rate_total)
        else:
            break_even_price = float('inf')
        
        # 4. 计算保本折扣率
        if payload.list_price > 0:
            break_even_discount_ratio = break_even_price / payload.list_price
        else:
            break_even_discount_ratio = float('inf')
        
        # 5. 判断是否亏损
        is_loss = net_profit < 0
        
        return ProfitResult(
            net_profit=round(net_profit, 4),
            net_margin=round(net_margin, 4),
            is_loss=is_loss,
            break_even_price=round(break_even_price, 4),
            break_even_discount_ratio=round(break_even_discount_ratio, 4)
        )
    
    @staticmethod
    def target_profit_price(target_profit: float, v: float, f: float) -> float:
        """目标利润定价"""
        return (target_profit + f) / (1 - v)
    
    @staticmethod
    def target_margin_price(target_margin: float, v: float, f: float) -> float:
        """目标利润率定价"""
        if target_margin >= 1 - v:
            raise ValueError(f"目标利润率不可达: 目标={target_margin}, 上限={1-v}")
        return f / (1 - v - target_margin)
    
    @staticmethod
    def target_roi_price(target_roi: float, v: float, f: float) -> float:
        """目标ROI定价"""
        return f * (1 + target_roi) / (1 - v)
    
    @staticmethod
    def simulate_discounts(payload: ProfitInput, ratios: List[float]) -> List[DiscountSimulation]:
        """折扣模拟"""
        results = []
        for ratio in ratios:
            # 计算折扣价
            deal_price = payload.list_price * ratio
            
            # 计算净利润
            net_profit = deal_price * (1 - payload.variable_rate_total) - payload.fixed_cost_total
            
            # 计算净利率
            net_margin = (net_profit / deal_price) if deal_price > 0 else 0.0
            
            # 判断是否亏损
            is_loss = net_profit < 0
            
            results.append(DiscountSimulation(
                discount_ratio=ratio,
                deal_price=round(deal_price, 4),
                net_profit=round(net_profit, 4),
                net_margin=round(net_margin, 4),
                is_loss=is_loss
            ))
        
        return results
```

```typescript
// ===== 类型定义 =====
interface ProfitInput {
  sale_price: number;
  list_price: number;
  variable_rate_total: number;
  fixed_cost_total: number;
}

interface ProfitResult {
  net_profit: number;
  net_margin: number;
  is_loss: boolean;
  break_even_price: number;
  break_even_discount_ratio: number;
}

interface DiscountSimulation {
  discount_ratio: number;
  deal_price: number;
  net_profit: number;
  net_margin: number;
  is_loss: boolean;
}

// ===== 利润求解器 =====
class ProfitSolver {
  /**
   * 计算当前利润
   */
  static solveCurrent(payload: ProfitInput): ProfitResult {
    // 1. 计算净利润
    const net_profit = payload.sale_price * (1 - payload.variable_rate_total) - payload.fixed_cost_total;
    
    // 2. 计算净利率
    const net_margin = payload.sale_price > 0 ? net_profit / payload.sale_price : 0;
    
    // 3. 计算保本价
    const break_even_price = payload.variable_rate_total < 1 
      ? payload.fixed_cost_total / (1 - payload.variable_rate_total)
      : Infinity;
    
    // 4. 计算保本折扣率
    const break_even_discount_ratio = payload.list_price > 0 
      ? break_even_price / payload.list_price
      : Infinity;
    
    // 5. 判断是否亏损
    const is_loss = net_profit < 0;
    
    return {
      net_profit: Math.round(net_profit * 10000) / 10000,
      net_margin: Math.round(net_margin * 10000) / 10000,
      is_loss,
      break_even_price: Math.round(break_even_price * 10000) / 10000,
      break_even_discount_ratio: Math.round(break_even_discount_ratio * 10000) / 10000
    };
  }
  
  /**
   * 目标利润定价
   */
  static targetProfitPrice(target_profit: number, v: number, f: number): number {
    return (target_profit + f) / (1 - v);
  }
  
  /**
   * 目标利润率定价
   */
  static targetMarginPrice(target_margin: number, v: number, f: number): number {
    if (target_margin >= 1 - v) {
      throw new Error(`目标利润率不可达: 目标=${target_margin}, 上限=${1-v}`);
    }
    return f / (1 - v - target_margin);
  }
  
  /**
   * 目标ROI定价
   */
  static targetRoiPrice(target_roi: number, v: number, f: number): number {
    return f * (1 + target_roi) / (1 - v);
  }
  
  /**
   * 折扣模拟
   */
  static simulateDiscounts(payload: ProfitInput, ratios: number[]): DiscountSimulation[] {
    return ratios.map(ratio => {
      const deal_price = payload.list_price * ratio;
      const net_profit = deal_price * (1 - payload.variable_rate_total) - payload.fixed_cost_total;
      const net_margin = deal_price > 0 ? net_profit / deal_price : 0;
      const is_loss = net_profit < 0;
      
      return {
        discount_ratio: ratio,
        deal_price: Math.round(deal_price * 10000) / 10000,
        net_profit: Math.round(net_profit * 10000) / 10000,
        net_margin: Math.round(net_margin * 10000) / 10000,
        is_loss
      };
    });
  }
}
```

### 5.3 SQL实现

```sql
-- ===== 1. 计算当前利润 =====
CREATE OR REPLACE FUNCTION calculate_profit(
  p_sale_price FLOAT,
  p_list_price FLOAT,
  p_variable_rate FLOAT,
  p_fixed_cost FLOAT
)
RETURNS TABLE (
  net_profit FLOAT,
  net_margin FLOAT,
  is_loss BOOLEAN,
  break_even_price FLOAT,
  break_even_discount_ratio FLOAT
) AS $$
BEGIN
  RETURN QUERY
  SELECT 
    ROUND((p_sale_price * (1 - p_variable_rate) - p_fixed_cost)::NUMERIC, 4)::FLOAT AS net_profit,
    ROUND(
      CASE 
        WHEN p_sale_price > 0 THEN (p_sale_price * (1 - p_variable_rate) - p_fixed_cost) / p_sale_price
        ELSE 0 
      END::NUMERIC, 
      4
    )::FLOAT AS net_margin,
    (p_sale_price * (1 - p_variable_rate) - p_fixed_cost) < 0 AS is_loss,
    ROUND(
      CASE 
        WHEN p_variable_rate < 1 THEN p_fixed_cost / (1 - p_variable_rate)
        ELSE NULL::FLOAT 
      END::NUMERIC, 
      4
    )::FLOAT AS break_even_price,
    ROUND(
      CASE 
        WHEN p_list_price > 0 AND p_variable_rate < 1 
        THEN (p_fixed_cost / (1 - p_variable_rate)) / p_list_price
        ELSE NULL::FLOAT 
      END::NUMERIC, 
      4
    )::FLOAT AS break_even_discount_ratio;
END;
$$ LANGUAGE plpgsql;

-- ===== 2. 目标利润定价 =====
CREATE OR REPLACE FUNCTION target_profit_price(
  p_target_profit FLOAT,
  p_variable_rate FLOAT,
  p_fixed_cost FLOAT
)
RETURNS FLOAT AS $$
BEGIN
  RETURN ROUND(((p_target_profit + p_fixed_cost) / (1 - p_variable_rate))::NUMERIC, 4);
END;
$$ LANGUAGE plpgsql;

-- ===== 3. 目标利润率定价 =====
CREATE OR REPLACE FUNCTION target_margin_price(
  p_target_margin FLOAT,
  p_variable_rate FLOAT,
  p_fixed_cost FLOAT
)
RETURNS FLOAT AS $$
BEGIN
  IF p_target_margin >= 1 - p_variable_rate THEN
    RAISE EXCEPTION '目标利润率不可达: % >= %', p_target_margin, 1 - p_variable_rate;
  END IF;
  
  RETURN ROUND((p_fixed_cost / (1 - p_variable_rate - p_target_margin))::NUMERIC, 4);
END;
$$ LANGUAGE plpgsql;

-- ===== 4. 目标ROI定价 =====
CREATE OR REPLACE FUNCTION target_roi_price(
  p_target_roi FLOAT,
  p_variable_rate FLOAT,
  p_fixed_cost FLOAT
)
RETURNS FLOAT AS $$
BEGIN
  RETURN ROUND((p_fixed_cost * (1 + p_target_roi) / (1 - p_variable_rate))::NUMERIC, 4);
END;
$$ LANGUAGE plpgsql;

-- ===== 5. 使用示例 =====
-- 计算当前利润
SELECT * FROM calculate_profit(100, 120, 0.35, 50);

-- 目标利润定价
SELECT target_profit_price(20, 0.35, 50);  -- 结果: 107.69

-- 目标利润率定价
SELECT target_margin_price(0.20, 0.35, 50);  -- 结果: 111.11

-- 目标ROI定价
SELECT target_roi_price(0.30, 0.35, 50);  -- 结果: 100.00
```

## 6. 边界条件处理

### 6.1 除零错误

**问题1**：`sale_price = 0` 时计算净利率

```python
# 错误写法

# 正确写法
net_margin = (net_profit / sale_price) if sale_price > 0 else 0.0
```

---

**问题2**：`variable_rate_total = 1` 时计算保本价

```python
# 错误写法
break_even_price = fixed_cost_total / (1 - variable_rate_total)  # ZeroDivisionError

# 正确写法
if variable_rate_total < 1:
    break_even_price = fixed_cost_total / (1 - variable_rate_total)
else:
    break_even_price = float('inf')  # 无法达到保本
```

---

**问题3**：`list_price = 0` 时计算保本折扣率

```python
# 错误写法
break_even_discount_ratio = break_even_price / list_price  # ZeroDivisionError

# 正确写法
if list_price > 0:
    break_even_discount_ratio = break_even_price / list_price
else:
    break_even_discount_ratio = float('inf')
```

### 6.2 目标利润率不可达

**问题**：目标利润率 + 变量费率 ≥ 1 时无解

```python
# 示例
variable_rate = 0.35
target_margin = 0.70  # 70%

# 检查
if target_margin >= 1 - variable_rate:  # 0.70 >= 0.65 → True
    raise ValueError("目标利润率不可达")

# 原因：
# 售价 = 固定成本 / (1 - 变量费率 - 目标利润率)
# 如果 1 - 变量费率 - 目标利润率 ≤ 0，则售价 = 负数或无穷大
```

## 8. 总结

### 8.1 核心公式汇总

| 算法 | 公式 | 用途 |
|------|------|------|
| **当前利润** | `净利润 = 售价 × (1 - 变量费率) - 固定成本` | 计算实际利润 |
| **保本价** | `保本价 = 固定成本 / (1 - 变量费率)` | 找到不亏本的价格 |
| **目标利润定价** | `售价 = (目标利润 + 固定成本) / (1 - 变量费率)` | 给定利润求售价 |
| **目标利润率定价** | `售价 = 固定成本 / (1 - 变量费率 - 目标利润率)` | 给定利润率求售价 |
| **目标ROI定价** | `售价 = 固定成本 × (1 + ROI) / (1 - 变量费率)` | 给定ROI求售价 |
| **折扣价** | `折扣价 = 原价 × 折扣率` | 模拟折扣效果 |
