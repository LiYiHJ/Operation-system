# 5 的详细步骤：Docker Desktop 一体化部署（Windows）

## 第 0 步：先确认你适不适合装

你适合走这条路的前提是：

- 你用的是 **Windows 10/11 64 位**，并且版本满足 Docker Desktop 要求。
- 机器开启了 **虚拟化**。
- 推荐使用 **WSL 2 后端**。
- Docker 官方要求 WSL 版本 **2.1.5 或更高**。([Docker Documentation](https://docs.docker.com/desktop/setup/install/windows-install/?utm_source=chatgpt.com))

先在 PowerShell 里检查：

```powershell
wsl --version
```

如果没装 WSL，或者版本太旧，先执行：

```powershell
wsl --install
wsl --update
```

这两条命令是 Docker 官方安装文档里推荐的 Windows 前置步骤。([Docker Documentation](https://docs.docker.com/installation/windows/?utm_source=chatgpt.com))

## 第 1 步：下载并安装 Docker Desktop

去 Docker 官方 Windows 安装页下载 **Docker Desktop Installer.exe**。官方说明里也给了安装器和安装方式。([Docker Documentation](https://docs.docker.com/desktop/setup/install/windows-install/?utm_source=chatgpt.com))

安装方式有两种。

### 图形安装

1. 双击 `Docker Desktop Installer.exe`
2. 安装时优先选择 **Use WSL 2 instead of Hyper-V**
3. 完成安装
4. 启动 Docker Desktop
5. 首次启动时接受服务协议

这些步骤都来自 Docker 官方安装说明。([Docker Documentation](https://docs.docker.com/desktop/setup/install/windows-install/?utm_source=chatgpt.com))

### 命令行安装

如果你想用 PowerShell：

```powershell
Start-Process 'Docker Desktop Installer.exe' -Wait install
```

这也是官方给出的安装方式。([Docker Documentation](https://docs.docker.com/desktop/setup/install/windows-install/?utm_source=chatgpt.com))

# Docker Desktop 到底是什么

你可以把它理解成：

**Windows 上运行 Docker 容器的官方桌面环境。**

它帮你解决这些事：

- 提供 Docker Engine
- 提供 `docker` 命令
- 提供 `docker compose`
- 管理镜像和容器
- 在 Windows 上通过 WSL 2 帮你跑 Linux 容器

因为你的项目里 PostgreSQL、Redis、后面可能还有 Web、Worker、Scheduler，最适合用 Docker Desktop 先统一跑起来。Docker 官方也明确推荐 Windows 上通过 WSL 2 或 Hyper-V 后端来使用 Docker Desktop。([Docker Documentation](https://docs.docker.com/desktop/setup/install/windows-install/?utm_source=chatgpt.com))
